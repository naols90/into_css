❓ What is CSS?

📝

HTML = Structure

Defines the content and layout of a webpage

Example: "This is a heading", "This is a paragraph"

🎨

CSS = Style

Defines how things LOOK

Example: "Make heading red", "Make paragraph larger"

✨ CSS stands for: Cascading Style Sheets
💡 Analogy: If HTML is the skeleton of a house, CSS is the paint, furniture, and decorations!

🤔 Why Do We Need CSS?

🚫

Without CSS

• Black text on white background

• Boring Times New Roman font

• No colors, no spacing

• Looks like a 1990s website

✅

With CSS

• Beautiful colors and backgrounds

• Any font you want

• Perfect spacing and alignment

• Professional, modern look

📌 Three Ways to Add CSS

1️⃣

Inline CSS

Inside HTML tags using style attribute

<p style="color:red;">Text</p>
2️⃣

Internal CSS

Inside <style> tag in the <head>

<style>
  p { color: red; }
</style>
3️⃣

External CSS

Separate .css file linked to HTML

<link rel="stylesheet" href="style.css">
💡 Best Practice: Use External CSS for real websites (keeps code organized)!

1️⃣ Inline CSS (Step by Step)

1 Create an HTML file:

<!DOCTYPE html>
<html>
<head>
    <title>My Page</title>
</head>
<body>
    <p>Hello World!</p>
</body>
</html>

2 Add style attribute to any tag:

<p style="color: blue;">Hello World!</p>

Result: This text is blue!

3 Add multiple styles:

<p style="color: blue; font-size: 24px; font-weight: bold;">Big Blue Bold Text</p>
💡 Separate multiple styles with semicolons (;)

2️⃣ Internal CSS (Step by Step)

1 Create an HTML file

2 Add <style> tag inside <head>

3 Write CSS rules inside

<!DOCTYPE html>
<html>
<head>
    <title>My Page</title>
    <style>
        p {
            color: blue;
            font-size: 20px;
        }
        h1 {
            color: red;
            text-align: center;
        }
    </style>
</head>
<body>
    <h1>Welcome!</h1>
    <p>This is a paragraph.</p>
    <p>Another paragraph.</p>
</body>
</html>
✅ Result: ALL paragraphs are blue and 20px. ALL h1 headings are red and centered!

📖 CSS Syntax Explained

selector {
    property: value;
    property: value;
}
🎯

Selector

The HTML element you want to style

p, h1, div, etc.
📦

Property

What you want to change

color, font-size, etc.
💎

Value

How you want to change it

red, 20px, bold, etc.
🔧

Rule

Selector + properties + values

p { color: red; }

3️⃣ External CSS (Step by Step)

1 Create a CSS file (e.g., style.css)

/* style.css */
body {
    background-color: #f0f0f0;
    font-family: Arial, sans-serif;
}

h1 {
    color: #1e3c72;
    text-align: center;
}

p {
    color: #333;
    font-size: 18px;
    line-height: 1.5;
}

2 Link it in your HTML file:

<!DOCTYPE html>
<html>
<head>
    <title>My Page</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <h1>Welcome!</h1>
    <p>This page uses external CSS!</p>
</body>
</html>

📁 External CSS - Folder Structure

Your project folder should look like this:

my_website/
│
├── index.html      (your HTML file)
├── style.css       (your CSS file)
└── images/         (optional - for images)
    └── logo.png

If CSS is in a different folder:

my_website/
├── index.html
└── css/
    └── style.css

Link it like this:

<link rel="stylesheet" href="css/style.css">

🎯 CSS Selectors - Element Selector

Element Selector

Selects all elements of a specific type

p {
    color: blue;
}

All <p> tags become blue

Example HTML

<p>First paragraph</p>
<p>Second paragraph</p>
<div>Not affected</div>

Try it:

This paragraph is blue (affected by p selector)

This paragraph is also blue

This div is NOT blue (not a paragraph)

🎯 CSS Selectors - Class Selector

Class Selector

Selects elements with a specific class attribute

.highlight {
    background: yellow;
    font-weight: bold;
}

Use a dot (.) before the class name

Example HTML

<p class="highlight">Highlighted</p>
<p>Normal text</p>
<div class="highlight">Also highlighted</div>

Try it:

This paragraph has class="highlight"

This paragraph has no class

This div also has class="highlight"
💡 Best Practice: Use classes to style multiple different elements the same way!

🎯 CSS Selectors - ID Selector

ID Selector

Selects ONE unique element

#header {
    background: #1e3c72;
    color: white;
    padding: 20px;
}

Use hash (#) before the ID name

ID must be unique on the page!

Example HTML

<div id="header">Header</div>
<p id="header">❌ Can't use same ID</p>
⚠️ Important: Each ID can only be used ONCE per page!

📊 Selector Types - When to Use Which?

Selector Syntax When to Use Example
Element element Style ALL elements of same type p { } (all paragraphs)
Class .classname Style MULTIPLE different elements the same way .error { } (all error messages)
ID #idname Style ONE specific unique element #logo { } (only the logo)
💡 Order of Specificity: ID > Class > Element (more specific overrides less specific)

💬 CSS Comments

Comments help you document your code (not visible on the webpage)

/* This is a CSS comment */
p {
    color: blue;  /* This comment explains the property */
}

/* 
   Multi-line comment
   Can span multiple lines
   Very useful for explaining complex code
*/
💡 Best Practice: Always comment your code! It helps others (and future you) understand what you did.

📦 Introduction to the Box Model

📦

Every HTML element is a box!

Every element has:

• Content (text, images, etc.)

• Padding (space inside border)

• Border (line around padding)

• Margin (space outside border)

div {
    width: 200px;
    padding: 20px;
    border: 2px solid black;
    margin: 30px;
}

Visual Representation:

Content Area

↑ Padding ↑

Border (dashed blue line)

↓ Margin (gray area) ↓

📦 Padding Property

Padding adds space INSIDE the element

/* All sides same */
padding: 20px;

/* Top/Bottom Left/Right */
padding: 10px 20px;

/* Top Right Bottom Left */
padding: 10px 15px 20px 25px;

/* Individual sides */
padding-top: 10px;
padding-right: 20px;
padding-bottom: 15px;
padding-left: 25px;

Live Example:

Padding: 10px all sides
Padding: 20px all sides (more space!)
💡 Padding increases the element's total size. A 200px box with 20px padding becomes 240px wide!

📦 Margin Property

Margin adds space OUTSIDE the element

/* All sides same */
margin: 20px;

/* Top/Bottom Left/Right */
margin: 10px 20px;

/* Top Right Bottom Left */
margin: 10px 15px 20px 25px;

/* Individual sides */
margin-top: 10px;
margin-right: 20px;
margin-bottom: 15px;
margin-left: 25px;

Live Example:

Margin: 10px
Margin: 30px (more space!)
💡 Auto margin for centering: margin: 0 auto; centers a block element horizontally!

📦 Border Property

/* Border styles */

🌐 Complete Example - Styling a Webpage

<!DOCTYPE html>
<html>
<head>
    <title>My Styled Page</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            background-color: #f5f5f5;
            margin: 0;
            padding: 20px;
        }
        
        h1 {
            color: #1e3c72;
            text-align: center;
            border-bottom: 3px solid #1e3c72;
            padding-bottom: 10px;
        }
        
        .card {
            background: white;
            border-radius: 10px;
            padding: 20px;
            margin: 20px 0;
            box-shadow: 0 2px 5px rgba(0,0,0,0.1);
        }
        
        .btn {
            background: #1e3c72;
            color: white;
            padding: 10px 20px;
            border: none;
            border-radius: 5px;
            cursor: pointer;
        }
        
        .btn:hover {
            background: #2a5298;
        }
    </style>
</head>
<body>
    <h1>Welcome to My Website</h1>
    <div class="card">
        <p>This is a styled card component.</p>
        <button class="btn">Click Me</button>
    </div>
</body>
</html>

⚠️ Common CSS Mistakes to Avoid

❌

Wrong

• Forgetting semicolons

• Misspelling property names

• Missing closing braces }

• Using spaces in values incorrectly

• Forgetting # for IDs or . for classes

✅

Correct

• color: red;

• background-color (not bgcolor)

• p { color: blue; }

• margin: 10px;

• #id or .class

Most Common: Forgetting the semicolon (;) between multiple properties!
